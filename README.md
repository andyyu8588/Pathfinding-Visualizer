# Pathfinding Visualizer
This is a desktop application to help programmers visualize different pathfinding algorithms. Built using C++, SFML, and TGUI.

### Featured algorithms:

- Dijkstra
- A*
- DFS
- BFS
- Bidirectional

Try it out in `./build/Pathfinding Visualizer.exe`
